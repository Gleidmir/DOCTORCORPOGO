# Códigos de Ativação do Meu Barbeiro GO

Guarde estes códigos para enviar aos seus clientes barbeiros após a confirmação do pagamento via Pix:

| Plano | Duração | Código de Ativação |
| :--- | :--- | :--- |
| **Mensal** | 30 dias | `ATIVA_MEN_MBG` |
| **Trimestral** | 90 dias | `ATIVA_TRI_MBG` |
| **Semestral** | 180 dias | `ATIVA_SEM_MBG` |
| **Anual** | 365 dias | `ATIVA_ANU_MBG` |
| **Vitalício / VIP** | Permanente | `MASTER_MBG_VIP` |

---

## Como usar:
1. Quando o barbeiro realizar o Pix e te enviar o comprovante, copie o código acima referente ao plano adquirido.
2. Envie o código para o barbeiro via WhatsApp.
3. O barbeiro irá acessar a tela de bloqueio do painel dele, colar o código no campo de ativação e clicar em **"Ativar Conta"**. O sistema irá validar o texto e liberar o acesso dele imediatamente.
